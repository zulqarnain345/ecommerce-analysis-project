E-Commerce Sales Data Analysis (Python Project)
Overview:
This project analyzes an e-commerce sales dataset to uncover business insights such as top-selling products, profit trends, and region-wise performance.
It uses Python, Pandas, Matplotlib, and Seaborn for data analysis and visualization.

Objectives:
Explore and summarize the e-commerce dataset
Identify top and least selling products
Analyze profit and sales trends over time
Compare sales performance across regions and categories
Visualize key insights using charts and heatmaps

Tools and Libraries Used:

Python 3
Pandas for data cleaning and analysis
Matplotlib for visualizations
Seaborn for statistical plots and heatmaps


Dataset:

File used: ecommerce_sales.csv

Sample columns:

Column	Description
Order_ID	Unique order identifier
Date	Order date
Product	Product name
Category	Product category
Quantity	Number of units sold
Price	Price per unit
Discount	Discount percentage
Profit	Profit earned
Total_Sales	Total revenue
Region	Sales region


Key Analyses Performed

Total number of categories and products
Total revenue, profit, and quantity sold
Top 5 best-selling and bottom 3 least-selling products
Region-wise and category-wise sales performance
Discount and profit relationship
Monthly and daily sales trends
Correlation between numerical features


Visualizations Included:
Bar Charts Region, Category, Product Prices

Pie Chart  Region-wise Sales Distribution

Line Charts:
Sales Trend Over Time
Profit Trend Over Time
Category-wise Average Sales
Region-wise Sales Trend
Monthly Sales Trend

Correlation Heatmap:
To show relationships between numerical columns


Insights Gained:

The highest-revenue category was identified using groupby analysis.
Found which region generated the most sales and which underperformed.
Visualized monthly sales trends and correlation between sales, profit, and discounts.
Helped understand which products should be promoted or discounted strategically.
